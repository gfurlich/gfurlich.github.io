Telescope Array Composite Satellite Map using Landsat Data

Date: Feb 8, 2020
Author: Greg Furlich

Tags : Telescope Array, Research, Remote Sensing, Python, LandSat 8, AWS, NumPy, GDAL,

The Telescope Array Cosmic Ray Observatory (TA) is the largest cosmic ray detector in the northern hemisphere. To better illustrate the observatory’s location and size for presentations and papers, I created my own composite satellite map. The python GDAL module was used to import the rasters of the different bands and NumPy (Numerical Python) was used to create a composite color image using the RGB bands. The satellite image was centered on the central laser facility (CLF) at the observatory. The Haversine formula was used to account for the relative great-circle distance between sites from the center given curvature of the earth. The satellite image below was created in this Jupyter python notebook.

Python and GDAL

The <a href”https://gdal.org/”>Geospatial Data Abstraction Library (GDAL)</a> is a library for working with the main forms of geospatial data (or images): rasters or vectors. The python GDAL module allows for importing raster and vector geospatial data in python.
