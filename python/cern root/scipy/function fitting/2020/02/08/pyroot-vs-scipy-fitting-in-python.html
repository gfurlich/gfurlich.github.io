CERN Root is a powerful statistical and fitting object oriented program that can be used in C++ by particle physicists. I use Root in my research in cosmic radiation given the amount of particle physicist in my community. However Root can now be imported into Python as Pyroot. I tested pyroot vs Scipy with fitting a random Gaussian histogram in a Jupyter python notebook.
