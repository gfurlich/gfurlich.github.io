  • Telescope Array Satellite Map using Landsat Data

  • February 08, 2020 • [ Telescope Array, Research, Remote Sensing, Python, LandSat 8, AWS, Numpy, GDAL, ]
  • Created a Satellite map of the Telescope Array Cosmic Ray Observatory located in the Sevier Desert of Millard County Utah using Landsat 8 data from AWS. The red, green, and blue bands were combined into a composite colored image using Numpy. Information was overlayed on the image. The curvature of the earth was taken into account using the Haversine formula.

  • Telescope Array Topography Maps using STRM Data

  • February 08, 2020 • [ Telescope Array, Research, Remote Sensing, Python, STRM, GDAL, ]
  • Created a topography map and shaded relief map of the Telescope Array Cosmic Ray Observatory located in the Sevier Desert of Millard County Utah using STRM data. STRM data was cropped to TA's location and an informational overlay was added. The curvature of the earth was taken into account using the Haversine formula.

  • Telescope Array Weather Classification and Other Interesting Pedestal Animations

  • November 10, 2019 • [ Telescope Array, Research, Machine Learning, Transient Luminous Events, ]
  • Research project to classify the weather in the Fluorescence Detector's field of view using machine learning. A series of snapshots of the night's progression using the baseline values of each photomultiplier tube and using it's nominal position on the night sky to reconstruct the detector's full field of view. These snapshots were feed into a Recurrent Convolution Neural Network (RCNN) for classification. Other interesting transient luminous events were seen in these snapshots.
